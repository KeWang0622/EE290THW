$$\min_{\boldsymbol x} \frac{1}{2} \|\boldsymbol y- \boldsymbol A \boldsymbol x\|_2^2 + \lambda \|\boldsymbol x\|_1 $$

(a) Implement Proximal Gradient for Lasso (sometimes called iterative shrinkage thresholding algorithm (ISTA)). This algorithm is given by:

Input: $\boldsymbol x_0 \in R^n$ and $L \geq \lambda_{max}(\boldsymbol A^* \boldsymbol A)$
while $\boldsymbol x_k$ not converged (k=1,2, ...) do

$\boldsymbol w_k \leftarrow \frac{1}{L}\boldsymbol A^* (\boldsymbol A \boldsymbol x_k -\boldsymbol y)$
$\boldsymbol x_{k+1} \leftarrow $ soft$(\boldsymbol w_k, \lambda/L)$

end while
Output: $\boldsymbol x_* \leftarrow \boldsymbol x_k$

def fft2c(x):
    return 1 / np.sqrt(np.prod(x.shape)) * np.fft.fftshift(np.fft.fft2(np.fft.ifftshift(x)))

def ifft2c(y):
    return np.sqrt(np.prod(y.shape)) * np.fft.ifftshift(np.fft.ifft2(np.fft.fftshift(y)))
